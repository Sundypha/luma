# Period Tracker PRD Package

This directory contains separate markdown PRDs for the phased delivery of a FOSS, privacy-first, local-first period tracker.

Files:
- `PRD_Phase_1_MVP.md`
- `PRD_Phase_2_Usability_and_Trust.md`
- `PRD_Phase_3_Secure_Sync.md`
- `PRD_Phase_4_Advanced_Prediction.md`

These documents are strict PRD-style artifacts focused on:
- problem framing,
- scope,
- requirements,
- rationale,
- acceptance criteria,
- pitfalls,
- anti-patterns,
- non-goals,
- release framing.
