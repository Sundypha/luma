**bayesian_algorithm.dart** — Create:

`BayesianAlgorithm implements PredictionAlgorithm`:
- Constructor: `const BayesianAlgorithm({this.priorMu = 28.0, this.priorKappa = 1.0, this.priorAlpha = 2.0, this.priorBeta = 15.0, this.defaultDuration = 5})`
- `id = AlgorithmId.bayesian`, `displayName = 'Pattern-learning'`, `minCycles = 2`
- `predict()`: Normal-Inverse-Gamma conjugate posterior update:
  - `κₙ = κ₀ + n`
  - `μₙ = (κ₀·μ₀ + n·x̄) / κₙ`
  - `αₙ = α₀ + n/2`
  - `βₙ = β₀ + ½·Σ(xᵢ - x̄)² + (κ₀·n·(x̄ - μ₀)²) / (2·κₙ)`
  - Point estimate: `μₙ.round()`. Anchor = last cycle start + predicted length.
  - Build explanation steps including posterior mean, prior info, observations count.

**linear_trend_algorithm.dart** — Create:

`LinearTrendAlgorithm implements PredictionAlgorithm`:
- Constructor: `const LinearTrendAlgorithm({this.rSquaredThreshold = 0.5, this.defaultDuration = 5})`
- `id = AlgorithmId.linearTrend`, `displayName = 'Trend'`, `minCycles = 5`
- `predict()`: Least-squares linear regression `y = slope * x + intercept` where x = cycle index.
  - Compute R² goodness-of-fit. If `R² < rSquaredThreshold`, return null (trend not significant).
  - Project: `projectedLength = (slope * n + intercept).round().clamp(18, 50)`.
  - Anchor = last cycle start + projected length.
  - Build explanation with slope, R², projection.
