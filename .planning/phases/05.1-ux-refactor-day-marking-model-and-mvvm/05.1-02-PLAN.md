    1. Normalize day to UTC midnight.
    2. Load all periods ordered by startUtc.
    3. Build `List<SpanRecord>` from stored periods (map id + startUtc + endUtc; for any remaining null endUtc â€” legacy â€” use startUtc as fallback).

    6. All within `_db.transaction()`.

    **`unmarkDay(DateTime day)`** â€” transactional method:
    1. Normalize day to UTC midnight.
    2. Load all periods ordered by startUtc.
    3. Build `List<SpanRecord>`.
    4. Call `computeUnmarkDay(spanRecords, day)`.
    5. Execute the resulting `DayUnmarkOp`:

       - `UnmarkNoOp`: Return success (day not marked).
    6. All within `_db.transaction()`.
