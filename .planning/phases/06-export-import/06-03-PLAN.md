  <action>
    **Add dependencies** to `apps/ptrack/pubspec.yaml`:
    - `share_plus: ^12.0.2`
    - `file_picker: ^11.0.0`

    Run `fvm flutter pub get` in workspace root.

    **Create `export_view_model.dart`** in `apps/ptrack/lib/features/backup/`:

    Define `enum ExportStep { selectContent, setPassword, exporting, done, error }`.

    `class ExportViewModel extends ChangeNotifier`:
    - Fields: `ExportStep _step`, `bool _includePeriods = true`, `bool _includeSymptoms = true`, `bool _includeNotes = true`, `String? _password`, `double _progress = 0`, `String? _errorMessage`, `ExportResult? _result`.
    - Getters for all fields.
    - `void applyPreset(ExportPreset preset)`: `everything` sets all true; `periodsOnly` sets only periods true, rest false. Notifies.
    - `void togglePeriods/toggleSymptoms/toggleNotes()`: flips boolean, notifies.
    - `void setPassword(String? pw)`: stores password, notifies.
    - `void nextStep()` / `void previousStep()`: navigate steps. Cannot go past `selectContent` if no content type selected.
    - `Future<void> startExport(ExportService service)`:
      1. Set step to `exporting`, progress to 0, notify.
      2. Build `ExportOptions` from current toggles + password.
      3. Call `service.exportData(options: options, onProgress: (c, t) { _progress = c / t; notifyListeners(); })`.
      4. Store result. Set step to `done`. Notify.
      5. On error: set step to `error`, store message. Notify.
    - `Future<void> deliverExport(BuildContext context)`:
      1. Write bytes to temp file using `getTemporaryDirectory()` + result.filename.
      2. On non-Linux: `SharePlus.instance.share(ShareParams(files: [XFile(tempPath)]))`.
      3. On Linux: `FilePicker.platform.saveFile(fileName: result.filename)` → write bytes to chosen path.
      4. Clean up temp file.
    - `void reset()`: resets all state to initial values.
    - `enum ExportPreset { everything, periodsOnly }`

    **Create `export_wizard_screen.dart`**:

    `class ExportWizardScreen extends StatefulWidget` — takes `ExportService service` in constructor.

    State creates `ExportViewModel` in initState, disposes in dispose.

    Build uses `ListenableBuilder(listenable: _vm, builder: ...)` switching on `_vm.step`:

    - **selectContent step**: Scaffold with:
      - Two `ChoiceChip` or `FilterChip` for presets ("Everything", "Periods only").
      - Three `SwitchListTile` for individual toggles (Periods, Symptoms & flow, Notes).
      - "Periods" switch disabled when it's the only one selected (must export something).
      - "Next" button → `_vm.nextStep()`.

    - **setPassword step**:
      - Explanatory text: "Optionally protect this backup with a password."
      - `TextField` for password (obscured) with clear button.
      - `TextField` for confirm password. Validate match on "Next".
      - "Skip" text button → proceeds without password.
      - "Next" button → `_vm.setPassword(...)` then `_vm.startExport(service)`.

    - **exporting step**:
      - `LinearProgressIndicator(value: _vm.progress)`.
      - Text showing percentage.
      - No back button — operation in progress.

    - **done step**:
      - Success icon + message.
      - Export metadata summary (date, content types, encrypted yes/no).
      - "Share" button → `_vm.deliverExport(context)`.
      - "Done" button → `Navigator.pop()`.

    - **error step**:
      - Error icon + message from `_vm.errorMessage`.
      - "Try Again" button → `_vm.reset()`.
      - "Close" button → `Navigator.pop()`.

    AppBar title: "Export Backup". Back arrow pops with confirmation if step > selectContent.

    **Tests** (`export_view_model_test.dart`):
    - Preset "everything" sets all toggles true.
    - Preset "periodsOnly" sets only periods true.
    - toggleSymptoms flips the value and notifies.
    - startExport transitions through exporting → done with correct result.
    - startExport with failing service transitions to error with message.
    - Password is passed through to ExportOptions correctly.
    - Cannot proceed from selectContent if all toggles are false.
