**Create `apps/ptrack/lib/features/calendar/calendar_day_data.dart`:**

1. `PeriodDayState` enum with values: `none`, `start`, `middle`, `end`, `single`, `middleRowStart`, `middleRowEnd`.
   - `start`: first day of a multi-day period (left-rounded, right-flat)
   - `middle`: interior day (full-width flat)
   - `end`: last day (left-flat, right-rounded)
   - `single`: one-day period (fully rounded)
   - `middleRowStart`: first day of a new week row that continues from a prior row (left-rounded, right-flat — visually restarts the band)
   - `middleRowEnd`: last day of a week row where the period continues into the next row (left-flat, right-rounded — visually caps the row)

2. `CalendarDayData` immutable class:
   ```dart
   @immutable
   class CalendarDayData {
     const CalendarDayData({
       this.loggedPeriodState = PeriodDayState.none,
       this.isPredictedPeriod = false,
       this.hasLoggedData = false,
       this.isToday = false,
     });
     final PeriodDayState loggedPeriodState;
     final bool isPredictedPeriod;
     final bool hasLoggedData;
     final bool isToday;
   }
   ```

3. `Map&lt;DateTime, CalendarDayData&gt; buildCalendarDayDataMap({required List&lt;StoredPeriodWithDays&gt; periodsWithDays, required PredictionResult prediction, required DateTime today, int startingDayOfWeek = DateTime.monday})`:

   **Step A — Build logged period day map.** For each period in `periodsWithDays`:
   - Normalize start and end to UTC calendar midnight: `DateTime.utc(d.year, d.month, d.day)`.
   - Walk each calendar day from startNorm to endNorm (inclusive). If period is open, walk up to today.
   - For each day, determine its raw `PeriodDayState`:
     - Single-day period (start == end): `single`
     - First day: `start`
     - Last day: `end`
     - Otherwise: `middle`
   - Then apply row-awareness: compute the day-of-week using `day.weekday`. The last day of a row is `(startingDayOfWeek + 6 - 1) % 7 + 1` (i.e., if Monday start, Sunday is last). The first day of a row is `startingDayOfWeek`. For middle days:
     - If it's the first day of the week row AND not the actual period start → `middleRowStart`
     - If it's the last day of the week row AND not the actual period end → `middleRowEnd`
   - Store in a `Map&lt;DateTime, PeriodDayState&gt;` keyed by UTC-midnight DateTime.

   **Step B — Mark predicted period days.** Extract predicted date range from `PredictionResult`:
   - `PredictionInsufficientHistory`: no predicted days
   - `PredictionRangeOnly(rangeStartUtc, rangeEndUtc)`: walk each day in range, mark as predicted
   - `PredictionPointWithRange(pointStartUtc, rangeStartUtc, rangeEndUtc)`: if range exists walk it; else just the point day
   - Only mark a day as `isPredictedPeriod: true` if it does NOT have a logged period state (logged data takes precedence over predictions).
   - Store in a `Set&lt;DateTime&gt;` (UTC-midnight keys).

   **Step C — Mark days with logged data.** Walk all `periodsWithDays[*].dayEntries[*].data.dateUtc`, normalize to UTC midnight, add to a `Set&lt;DateTime&gt;`.

   **Step D — Assemble.** Merge all three collections + today into a single `Map&lt;DateTime, CalendarDayData&gt;`:
   ```dart
   final todayNorm = DateTime.utc(today.year, today.month, today.day);
   final allDays = {...periodDayStates.keys, ...predictedDays, ...loggedDataDays, todayNorm};
   return {
     for (final day in allDays)
       day: CalendarDayData(
         loggedPeriodState: periodDayStates[day] ?? PeriodDayState.none,
         isPredictedPeriod: predictedDays.contains(day),
         hasLoggedData: loggedDataDays.contains(day),
         isToday: day == todayNorm,
       ),
   };
   ```

   Import `StoredPeriodWithDays` from `ptrack_data`, `PredictionResult` types from `ptrack_domain`, `package:meta/meta.dart`.

**1. PeriodBandPainter** — draws a solid-fill horizontal band segment based on `PeriodDayState`:
- Constructor: `PeriodBandPainter({required PeriodDayState state, Color color = kPeriodColor})`
- `paint(Canvas canvas, Size size)`:
  - Band height = `size.height * 0.52` (centered vertically)
  - `top = (size.height - bandHeight) / 2`
  - Base rect = `Rect.fromLTWH(0, top, size.width, bandHeight)`
  - Radius = `Radius.circular(bandHeight / 2)`
  - Build `RRect` based on state:
    - `start`: left corners rounded, right corners square
    - `end`: right corners rounded, left corners square
    - `single`: all corners rounded
    - `middle`: no rounding (Radius.zero)
    - `middleRowStart`: same as `start` (left-rounded, right-flat)
    - `middleRowEnd`: same as `end` (left-flat, right-rounded)
    - `none`: don't paint
  - `canvas.drawRRect(rrect, Paint()..color = color)`
- `shouldRepaint`: compare `state` and `color`.

**2. HatchedCirclePainter** — draws a circle with diagonal stripe fill for predicted days:
- Constructor: `HatchedCirclePainter({Color color = kPeriodColor, double stripeSpacing = 4.0})`
- `paint(Canvas canvas, Size size)`:
  - `center = Offset(size.width / 2, size.height / 2)`
  - `radius = size.shortestSide * 0.36`
  - Create circle path: `Path()..addOval(Rect.fromCircle(center: center, radius: radius))`
  - `canvas.save(); canvas.clipPath(circlePath);`
  - Draw diagonal stripes (bottom-left to top-right) using `Paint()..color = color..strokeWidth = 1.5..style = PaintingStyle.stroke`:
    ```dart
    for (double offset = -size.height; offset < size.width + size.height; offset += stripeSpacing) {
      canvas.drawLine(Offset(offset, size.height), Offset(offset + size.height, 0), stripePaint);
    }
    ```
  - `canvas.restore();`
  - Draw circle outline: `canvas.drawCircle(center, radius, Paint()..color = color..style = PaintingStyle.stroke..strokeWidth = 1.5)`
- `shouldRepaint`: compare `color` and `stripeSpacing`.

**3. DotIndicatorPainter** — small dot below center for days with logged data:
- Constructor: `DotIndicatorPainter({Color color = kPeriodColor})`
- `paint`: draw a filled circle at `Offset(size.width / 2, size.height * 0.82)` with radius 2.5.
- `shouldRepaint`: compare `color`.

**4. TodayRingPainter** — ring/outline around today's date:
- Constructor: `TodayRingPainter({Color color = kPeriodColor})`
- `paint`: draw a circle stroke at center with `radius = size.shortestSide * 0.38`, `strokeWidth = 2.0`, `PaintingStyle.stroke`.
- `shouldRepaint`: compare `color`.

**5. Helper: `buildDayCell` function** (or keep in calendar_screen.dart — either is fine, but providing a helper here reduces Plan 03 complexity):
```dart
Widget buildCalendarDayCell(DateTime day, CalendarDayData data) {
  return Stack(
    alignment: Alignment.center,
    children: [
      if (data.loggedPeriodState != PeriodDayState.none)
        CustomPaint(painter: PeriodBandPainter(state: data.loggedPeriodState), size: Size.infinite),
      if (data.isPredictedPeriod && data.loggedPeriodState == PeriodDayState.none)
        CustomPaint(painter: HatchedCirclePainter(), size: Size.infinite),
      if (data.isToday)
        CustomPaint(painter: TodayRingPainter(), size: Size.infinite),
      Text(
        '${day.day}',
        style: TextStyle(
          color: data.loggedPeriodState != PeriodDayState.none ? Colors.white : null,
          fontWeight: data.isToday ? FontWeight.bold : null,
        ),
      ),
      if (data.hasLoggedData)
        CustomPaint(painter: DotIndicatorPainter(), size: Size.infinite),
    ],
  );
}
```
