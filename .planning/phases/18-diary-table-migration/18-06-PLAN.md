```dart
Future<void> _openDiaryFormAfterPop({
  required BuildContext sheetContext,
  required CalendarViewModel viewModel,
  required DateTime dayNorm,
  StoredDiaryEntry? existing,
}) async {
  final nav = Navigator.of(sheetContext);
  final overlay = nav.overlay;
  nav.pop();
  if (overlay == null) return;
  WidgetsBinding.instance.addPostFrameCallback((_) {
    if (!overlay.context.mounted) return;
    showDiaryFormSheet(
      overlay.context,
      diaryRepository: viewModel.diaryRepository,
      day: dayNorm,
      existing: existing,
    );
  });
}
```

  - Empty state: center text "No tags yet. Add your first tag below." with a `+` button.

**Tag list tile** for each tag:
- `ListTile` with `title: Text(tag.name)` and two trailing action icons:
  - **Edit icon** (`Icons.edit_outlined`): opens a rename dialog (see below).
  - **Delete icon** (`Icons.delete_outline`): calls `diaryRepository.entryCountForTag(tag.id)` then shows `AlertDialog`:
    - If count > 0: "This tag is used in {n} entries. Removing it will not delete those entries."
    - Confirm → `diaryRepository.deleteTag(tag.id)`

**Rename dialog** (shown by edit icon tap):
```dart
showDialog<void>(
  context: context,
  builder: (_) {
    final controller = TextEditingController(text: tag.name);
    return AlertDialog(
      title: Text(l10n.diaryTagsRenameTitle),
      content: TextField(
        controller: controller,
        autofocus: true,
        decoration: InputDecoration(labelText: l10n.diaryTagsAddLabel),
      ),
      actions: [
        TextButton(
          onPressed: () => Navigator.of(context).pop(),
          child: Text(l10n.cancel),
        ),
        FilledButton(
          onPressed: () async {
            final name = controller.text.trim();
            if (name.isEmpty || name == tag.name) {
              Navigator.of(context).pop();
              return;
            }
            await diaryRepository.renameTag(tag.id, name);
            if (context.mounted) Navigator.of(context).pop();
          },
          child: Text(l10n.diaryTagsRenameConfirm),
        ),
      ],
    );
  },
);
```

**Add tag row** (pinned at bottom via `Column` with `Spacer` or `ListView` footer):
- `TextField` + `FilledButton.tonal('Add')` in a `Row`
- On submit: `diaryRepository.createTag(value)` → clear field → Snackbar "Tag added"
- On error (empty or duplicate): Snackbar with error message
