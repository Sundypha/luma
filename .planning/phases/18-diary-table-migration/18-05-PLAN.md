**`home_view_model.dart`** — add diary awareness:

1. Add `DiaryRepository diaryRepository` to the HomeViewModel constructor (required).
2. Add `StoredDiaryEntry? _todayDiaryEntry;` private field.
3. Subscribe to `diaryRepository.watchAllEntries()` in init; find the entry matching today's UTC midnight:
   ```dart
   _diarySub = diaryRepository.watchAllEntries().listen((entries) {
     final today = _utcCalendarDay(DateTime.now());
     _todayDiaryEntry = entries.cast<StoredDiaryEntry?>().firstWhere(
       (e) => e?.data.dateUtc == today,
       orElse: () => null,
     );
     notifyListeners();
   });
   ```
4. Expose: `StoredDiaryEntry? get todayDiaryEntry => _todayDiaryEntry;`
5. Cancel sub in dispose.

**`today_card.dart`** — add diary shortcut row:

Below the existing period status section (after whatever shows the current day status), add a diary shortcut row. It should always be visible (not just on period days):

```dart
// Diary shortcut row
ListTile(
  leading: const Icon(Icons.book_outlined),
  title: Text(
    viewModel.todayDiaryEntry != null
        ? l10n.homeDiaryEditEntry
        : l10n.homeDiaryNewEntry,
  ),
  subtitle: viewModel.todayDiaryEntry?.data.notes != null
      ? Text(
          viewModel.todayDiaryEntry!.data.notes!,
          maxLines: 1,
          overflow: TextOverflow.ellipsis,
        )
      : null,
  trailing: const Icon(Icons.chevron_right),
  onTap: () => showDiaryFormSheet(
    context,
    diaryRepository: viewModel.diaryRepository,
    day: _utcCalendarDay(DateTime.now()),
    existing: viewModel.todayDiaryEntry,
  ),
),
```
