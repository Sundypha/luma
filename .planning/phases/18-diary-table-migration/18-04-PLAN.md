The sheet is a stateful modal bottom sheet opened via `showDiaryFormSheet(context, ...)`.

**Constructor parameters:**
```dart
class DiaryFormSheet extends StatefulWidget {
  const DiaryFormSheet({
    super.key,
    required this.diaryRepository,
    required this.day,         // UTC midnight — the day being edited
    this.existing,             // StoredDiaryEntry if editing, null if creating
  });

  final DiaryRepository diaryRepository;
  final DateTime day;
  final StoredDiaryEntry? existing;
}
```

**Opener function:**
```dart
Future<void> showDiaryFormSheet(
  BuildContext context, {
  required DiaryRepository diaryRepository,
  required DateTime day,
  StoredDiaryEntry? existing,
}) {
  return showModalBottomSheet<void>(
    context: context,
    isScrollControlled: true,
    showDragHandle: true,
    useSafeArea: true,
    builder: (_) => DiaryFormSheet(
      diaryRepository: diaryRepository,
      day: day,
      existing: existing,
    ),
  );
}
```

**State fields:**
```dart
late TextEditingController _notesController;
Mood? _mood;
List<DiaryTag> _allTags = [];
Set<int> _selectedTagIds = {};
StreamSubscription<List<DiaryTag>>? _tagsSub;
bool _saving = false;
```

**initState:** populate from `existing` if present; subscribe to `diaryRepository.watchTags()` to populate `_allTags`.

**UI layout** (`SingleChildScrollView` + `padding: EdgeInsets.only(bottom: MediaQuery.viewInsetsOf(context).bottom + 16)`):
1. Date title (full date, same style as day_detail_sheet.dart).
2. **Notes field:** `TextField` with `maxLines: null`, `minLines: 3`, `keyboardType: TextInputType.multiline`, `decoration: InputDecoration(labelText: l10n.diaryNotesLabel, border: OutlineInputBorder())`.
3. **Mood selector:** Same discrete slider pattern as symptom_form_sheet.dart (`_DiscreteSymptomSlider` or similar). Label: `l10n.diaryMoodLabel`. Use `Mood` enum values 1–5; 0 = not set.
4. **Tag section:** Label `l10n.diaryTagsLabel` above a `Wrap` of `FilterChip` widgets for each tag in `_allTags` (selected if `_selectedTagIds.contains(tag.id)`; on selected toggle `_selectedTagIds`).
5. **Inline tag creation:** Below the chips, a small `TextField` with `decoration: InputDecoration(labelText: l10n.diaryAddTagLabel, suffixIcon: Icon(Icons.add))`. On submit (onFieldSubmitted / icon tap): call `diaryRepository.createTag(value)`, the stream updates `_allTags`, auto-select the new tag. Clear the field.
6. **Action row at bottom:**
   - `FilledButton(l10n.commonSave)` → calls `_save()`.
   - If `existing != null`: `TextButton(l10n.commonDelete, color: error)` → confirms deletion → `diaryRepository.deleteEntry(existing!.id)` → `Navigator.pop`.

**`_save()`:**
```dart
final data = DiaryEntryData(
  dateUtc: widget.day,
  mood: _mood,
  notes: _notesController.text.trim().isEmpty ? null : _notesController.text.trim(),
);
await widget.diaryRepository.saveEntry(data, tagIds: _selectedTagIds.toList());
Navigator.of(context).pop();
```
