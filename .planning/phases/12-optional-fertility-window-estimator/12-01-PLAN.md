## Model

`FertileWindow` — immutable value object:
- `startUtc` (DateTime): first fertile day (UTC midnight)
- `endUtc` (DateTime): last fertile day (UTC midnight)
- `estimatedOvulationUtc` (DateTime): estimated ovulation day (UTC midnight)

## Static methods on `FertilityWindowCalculator`

### `compute`
```
FertileWindow? compute({
  required DateTime lastPeriodStartUtc,
  required int cycleLengthDays,
  int lutealPhaseDays = 14,
})
```

**Formula (standard calendar method):**
- Estimated ovulation day = `cycleLengthDays - lutealPhaseDays` days after `lastPeriodStartUtc`
- Fertile window start = ovulation day − 5 (sperm survival window)
- Fertile window end = ovulation day + 1 (egg viability ~24 h)
- Total window: 7 days

**Constraints:**
- `cycleLengthDays` must be ≥ 10 and ≤ 60; return null otherwise
- `lutealPhaseDays` must be ≥ 5 and ≤ 20; return null otherwise
- If ovulation day ≤ 5 (fertile start would be before period start), clamp fertile start to day 1

**Document in a doc comment:** the formula, the assumption defaults (14-day luteal), the clinical basis (sperm survival up to 5 days, ovum viability ~12-24 h), and the educational-estimate-only caveat.

### `averageCycleLengthFromHistory`
```
int? averageCycleLengthFromHistory(List<int> cycleLengths)
```
- Empty list → null
- Non-empty → arithmetic mean, rounded to nearest int
